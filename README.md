 Smart Travel Planner – Key Points
Project Objective
To simplify travel planning by generating personalized trip itineraries.
Problem Addressed
Eliminates the need to use multiple platforms for itinerary, budget, and weather planning.
User Inputs
Destination, number of days, number of people, budget, and type of trip.
Core Functionality
Automatically generates a day-wise travel plan with places and timings.
Budget Planning
Provides estimated cost breakdown for travel, food, accommodation, and activities.
Weather Integration
Uses API to display real-time weather and forecast of the destination. 
Safety Features
Suggests precautions and travel tips for a safe journey.
Technologies Used
HTML, CSS, JavaScript, Node.js, npm, Figma, GitHub, Visual Studio Code.
Advantages
Saves time, provides personalized planning, and offers an all-in-one solution.
Future Scope
Can be enhanced with AI recommendations, booking systems, maps, and chatbot integration.
